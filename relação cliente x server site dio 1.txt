<p>Os clientes são os usuários do site, que acessam ele por meio de seu computador ou celular, colocando um link para encontrar o site e pedir informações. o papel do servidor, o lugar onde você hospeda o site, é enviar informações e arquivos para o cliente. O servidor pode ser menor, sendo dentro de seu próprio computador, ou maior, como em um dos servidores do Google, que possui uma quantidade gigantesca de servidores.</p>

<p>O cliente pode estar em vários dispositivos diferentes, como em um computador, notebook, tablet, celular... Por este motivo, é muito importante o site ser compatível com vários desses dispositivos. Todo site é acessado por meio da internet, que faz a conexão entre a máquina do cliente e o servidor do site, para que o servidor envie as informações e arquivos de volta para o cliente.</p>
<br><br>
<p>O servidor pode armazenar conteúdos como textos, imagens, arquivos, áudios, entre outros. O servidor é onde o site armazena essas informações e arquivos, esperando algum cliente acessar o site para enviá-las para o cliente.</p>
<p>Os navegadores são programas criados por empresas feitos para executar arquivos de sites. Entre eles, os mais famosos com certeza são o <strong>Google Chrome</strong>, <strong>Internet Explorer</strong>, <strong>Ópera GX</strong>, entre outros.. Esses navegadores possuem funcionalidades e interpretações diferentes, mas todos seguem o básico do padrão W3C. Os primeiros navegadores foram o MOSAIC e o Netscape, a partir desses, outros foram criados e melhoraram o jeito das pessoas de navegar na internet. Completamente gratuitos, servem como um meio para a população (clientes) acesar os sites (servidores).</p>
<p>As aplicações WEB são sites feitos para que os clientes não precisem baixar um aplicativo para usar aquele produto. Por exemplo, o YouTube tem seu aplicativo, que é usado dispositivos móveis, em sua maioria. Porém, o YouTube também tem seu site, que pode ser usado sem baixar. Neste exemplo, o aplicativo do YouTube não é uma aplicação Web. Já o site do YouTube, esse sim é uma aplciação Web. Muitos sites tem aplicativos ou vice-versa, como o Twitter, Facebook, LinkedIn, etc..</p>
